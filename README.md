# Cryptography Projects

This repository contains implementations of classical cryptographic algorithms and cryptanalysis techniques developed in Python as part of university coursework.

## Projects included

### 1. Vigenère Cipher Cryptanalysis
Implementation of encryption, decryption and statistical cryptanalysis of the Vigenère cipher.

Features:
- Vigenère encryption and decryption
- Index of Coincidence (IC) calculation
- Key length estimation
- Chi-square frequency analysis
- Automatic key recovery for English text

---

### 2. Simplified DES (SDES)
Implementation of the Simplified DES block cipher with two modes of operation.

Features:
- Complete SDES encryption and decryption
- Key scheduling algorithm
- ECB (Electronic Codebook) mode
- CBC (Cipher Block Chaining) mode

---

### 3. Diffie–Hellman Key Exchange
Implementation of the Diffie–Hellman protocol for secure key exchange.

Features:
- Miller–Rabin primality test
- Safe prime generation
- Primitive root selection
- Shared secret computation

---

## Technologies
- Python 3
- Basic number theory
- Classical cryptography concepts

---

## Purpose
The goal of these projects is to understand how classical cryptographic algorithms work internally and how statistical and mathematical methods can be used in cryptanalysis.
